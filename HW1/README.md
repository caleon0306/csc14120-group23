## Quick Start
Extract this Directory into your virtual enviroment area

Start the Virtual Enviroment:
```
source bin/activate
```

Ensure the requierments are installed using:
```
pip3 install -r csci4120-nogroup-leonc22-main/HW1/requirements.txt
```
